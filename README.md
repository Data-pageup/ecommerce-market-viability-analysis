# E-commerce Market Viability Analysis

This project evaluates e-commerce market viability and operational risks
using transactional, customer, and seller data.

## Objectives
- Analyze customer behavior and demand patterns
- Assess operational risk from orders, payments, and reviews
- Identify factors impacting market stability

## Dataset
Public multi-table retail dataset including customers, orders,
payments, reviews, products, and sellers.

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn
